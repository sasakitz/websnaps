# WebSnaps

Webページのスクリーンショットを撮影・編集できるChrome拡張機能です。

## 機能

### キャプチャモード

| モード | 説明 | ショートカット |
|--------|------|---------------|
| 範囲選択 | ドラッグまたはDOM要素を選択してキャプチャ | `Alt+Shift+S` |
| 表示範囲 | 現在表示されているビューポートをキャプチャ | `Alt+Shift+V` |
| ページ全体 | スクロールを含むページ全体をキャプチャ | `Alt+Shift+F` |

> ショートカットは `chrome://extensions/shortcuts` から変更可能です。

### 範囲選択モードの操作

- **矩形選択** — ドラッグして任意の矩形範囲を指定
- **要素選択** — DOM要素にホバーしてクリックで選択
- `Esc` でキャンセル

### エディタ機能

キャプチャ後に編集画面が開きます。

| ツール | 説明 |
|--------|------|
| ペン | フリーハンド描画 |
| 直線 / 矢印 | 直線・矢印の描画 |
| 四角形 / 楕円 | 図形の描画（塗りつぶし対応） |
| テキスト | 文字の入力 |
| ぼかし | 指定範囲をぼかす |
| モザイク | 指定範囲をモザイク処理 |
| 消しゴム | 描画内容を消去 |
| 切り抜き | 指定範囲に画像をトリミング |

### 保存オプション

- **PNG / JPG** でファイル保存
- **クリップボードにコピー**

## インストール

1. このリポジトリをクローンまたはZIPでダウンロード
   ```
   git clone https://github.com/sasakitz/websnaps.git
   ```
2. Chromeで `chrome://extensions` を開く
3. 右上の「デベロッパーモード」をオンにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. ダウンロードしたフォルダを選択

## アイコンの再生成

アイコンPNGを再生成する場合:

```bash
python3 icons/generate_icons.py
```

## 動作環境
