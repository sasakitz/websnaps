## 詳細説明

```
WebSnaps は、Webページのスクリーンショット撮影と編集をシームレスに行える Chrome 拡張機能です。

─────────────────────────────────────
🎯 3種類のキャプチャモード
─────────────────────────────────────

■ 範囲選択キャプチャ
ドラッグで任意の矩形範囲を選択するか、DOM要素にホバーしてクリックするだけで選択できます。選択後はハンドルをドラッグしてサイズを微調整したり、W/H を数値で直接指定することも可能です。

■ 表示範囲キャプチャ
現在ブラウザに表示されている範囲をそのままキャプチャします。

■ ページ全体キャプチャ
スクロールが必要な長いページも、自動スクロールしながら全体を繋ぎ合わせて1枚の画像として保存します。

─────────────────────────────────────
✏️ 内蔵エディタで即座に編集
─────────────────────────────────────

キャプチャ後に自動でエディタが開き、そのまま編集できます。

• ペン — フリーハンド描画
• 直線 / 矢印 — 強調したい箇所への矢印付き
• 四角形 / 楕円 — 囲み線・ハイライト
• テキスト — 注釈・コメントの書き込み
• ぼかし — 個人情報・機密情報のマスク
• モザイク — 強めのモザイク処理
• 切り抜き — 不要な周囲をトリミング
• 消しゴム — 描いた内容を部分消去
• 塗りつぶし — 図形の塗りつぶし ON/OFF

無制限のアンドゥ・リドゥで気軽に試せます。

─────────────────────────────────────
💾 柔軟な保存オプション
─────────────────────────────────────

• PNG / JPG を選択して PC に保存
• クリップボードにコピーしてそのまま貼り付け

─────────────────────────────────────
⌨️ ショートカットキー対応
─────────────────────────────────────

• 範囲選択: Alt+Shift+S
• 表示範囲: Alt+Shift+V
• ページ全体: Alt+Shift+F

chrome://extensions/shortcuts からご自身のキーに変更できます。

─────────────────────────────────────
🔒 プライバシーについて
─────────────────────────────────────

すべての処理はお使いのブラウザ内でのみ行われます。スクリーンショットデータが外部サーバーに送信されることは一切ありません。
```

## English Description（英語説明）

```
WebSnaps is a Chrome extension for capturing and editing screenshots of web pages.

── 3 Capture Modes ──

■ Area Selection
Drag to select any rectangular area, or hover over DOM elements to select them precisely. After selection, drag the resize handles or enter exact W/H values to fine-tune the area.

■ Visible Area
Capture exactly what is currently visible in your browser.

■ Full Page
Automatically scrolls through the entire page and stitches it into a single image.

── Built-in Editor ──

The editor opens automatically after capture:
• Pen, Line, Arrow, Rectangle, Ellipse
• Text annotations
• Blur / Mosaic for hiding sensitive info
• Crop, Eraser
• Unlimited undo / redo

── Save Options ──
• Save as PNG or JPG
• Copy to clipboard

── Keyboard Shortcuts ──
Alt+Shift+S / V / F (customizable)

── Privacy ──
All processing is done locally in your browser. No data is ever sent to external servers.
```
